## cd

### 概述
cd命令也是用的最多的一个命令,英文意思为change directory-改变目录 功能是切换工作目录


### 解析

|   命令  |  英文释义   |    参数   |
|--|--|--|
|    cd   | change directory | . .. path|


### 实例

```bash
cd .
# .为当前目录 则目录不改变

cd ..
# ..为上一级目录 工作目录切换到上一级目录

cd -
# 切换为系统环境变量OLDPATH所在的目录 一般为上一次的工作目录


cd 
cd ~
# cd不带参和参数为~功能一致,则直接切换为login时的目录

```
